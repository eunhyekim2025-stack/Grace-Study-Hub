# System Variable — Contract Review Unit Cost

As adoption rises, the unit cost falls (`dampens` from [[systems/ai-adoption]]),
and a lower unit cost in turn drives adoption back up (`dampens` → ai-adoption).
Two negative (−) links → **reinforcing (R) loop** (can run in either direction: virtuous or vicious cycle).
