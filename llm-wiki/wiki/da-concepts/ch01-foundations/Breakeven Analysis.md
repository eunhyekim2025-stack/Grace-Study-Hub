# Breakeven Analysis

A cost-volume-profit model used to find the **breakeven point** — the production volume at which total revenue = total cost.

## Basic Model

| Model | Equation |
|------|-----|
| Total Cost | `C(x) = FC + v·x` (fixed cost + unit variable cost × quantity) |
| Total Revenue | `R(x) = p·x` (price × quantity) |
| Profit | `P(x) = R(x) − C(x) = (p − v)·x − FC` |

- **Unit contribution margin** = `p − v`
- **Breakeven point**: `R(x) = C(x)` → `x = FC / (p − v)`

## Meaning
- Sales volume > breakeven point → profit
- Sales volume < breakeven point → loss
- The larger the contribution margin, the lower the breakeven point
